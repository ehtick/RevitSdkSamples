Export options

 
 
 

In the Export Options dialog, you can define export parameters. The dialog displays when you click Options in the Export section of the Options dialog.

Define export parameters:

  • Layers and properties
    • By Entity
    • By Layer
    • New Layer
  • Export Layers Standard
    • Default
    • AIA
    • CP38
    • BS1192
    • ISO13567
  • Linetype scaling
    • Model space
    • Paper space
    • View scale
  • Coordinate system bases
    • Project internal
    • Shared
  • Units of a DWG file
    • Millimeters
    • Centimeters
    • Meters
    • Inches
    • Feet
  • Export rooms and areas as polylines
