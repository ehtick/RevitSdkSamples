Options

 
 
 

Use the Options dialog to select parameters applied while freezing drawings/views. The dialog displays when you click Options on the Freeze Drawings dialog.

Define parameters:

  • Under Export, click Options to open the Export Options dialog.
  • Under View, select Remove the source view to delete the source views of frozen views/drawings.
  • Under Import colors, select the colors to be applied while importing a DWG file:
    • Black and white
    • Preserve colors
    • Invert colors
  • Under Files, define all parameters necessary to save a DWG file.
