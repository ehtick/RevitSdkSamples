Freeze drawings dialog

 
 
 

After you launch the Freeze Drawings extension, the Freeze drawings dialog displays.

In the dialog, you select the range of drawings / views to freeze:

  • Current view - freezes only the active view.
  • Selected views - allows you to click Select, which opens the Selected views dialog.
NoteFreezing of drawings does not include 3D views or sheets.

At the bottom of the Freeze Drawings dialog, there are 3 buttons:

  • Options - click this to open the Options dialog, where you can define parameters used while freezing drawings / views.
  • OK - click this to start the freeze process.
  • Cancel - click this to close the Freeze Drawings extension.
